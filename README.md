# top_library

A simple book library app where you can add, delete, and track books you've read.

## Features

- ✅ Add new books with title, author, pages, and read status
- ✅ Toggle read/not read status
- ✅ Delete books from library

## How to Use

1. Enter book details in the "Add New Book" form
2. Select "Read" or "Not Read" status
3. Click "Add Book"
4. Use "Toggle Read" to change read status
5. Use "Delete" to remove books

## Tech Stack

- HTML5
- CSS3 (Flexbox/Grid)
- JavaScript
